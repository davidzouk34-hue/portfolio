## 📝 Présentation du Projet
Dans un contexte où la présence numérique est devenue indispensable pour les entreprises, l'agence digitale **IgoTech** souhaite mettre en place un portfolio en ligne afin de présenter ses activités, ses réalisations et ses compétences. Ce portfolio sert de vitrine numérique permettant à l'agence d'améliorer sa visibilité, de valoriser ses projets et d'attirer de nouveaux clients.

### 🎯 Objectifs
* **Présenter** l'agence et son équipe.
* **Mettre en valeur** les réalisations (projets) effectuées.
* **Exposer** la palette complète de services proposés.
* **Partager** les témoignages et retours de clients satisfaits.
* **Faciliter la prise de contact** via un formulaire ergonomique.
* **Optimiser la visibilité** de l'agence sur le web.

## 🛠️ Technologies Utilisées
Le projet a été conçu en respectant les standards actuels du développement web :
* **HTML5** : Structuration sémantique et claire du site.
* **CSS3** : Mise en forme, responsive design (adaptabilité mobile, tablette, PC) et transitions fluides.
* **Git & GitHub** : Versioning et hébergement du code source.
* **VS Code** : Éditeur de code principal utilisé pour le développement.

## 📐 Architecture du Site
Le portfolio est organisé selon la structure modulaire suivante :
```text
Accueil
│
├── À propos
│
├── Services
│   ├── Création de sites web
│   ├── Gestion des réseaux sociaux
│   ├── Création de contenu visuel
│   └── Marketing digital
│
├── Portfolio (Réalisations)
│   ├── Projet 1 (Site web e-commerce)
│   ├── Projet 2 (Application de gestion)
│   └── Projet 3 (Campagne marketing digitale)

├── Témoignages
│
└── Contact (Formulaire)
```

---

## 🎨 Aperçu de l'Interface & Maquette
Le design a été pensé pour être **moderne, épuré et professionnel**, avec des sections bien délimitées et une palette de couleurs basée sur le bleu et le blanc pour inspirer la confiance et l'innovation technologique.

### 🌟 Sections Principales :
1. **En-tête & Navigation** : Navigation fluide avec liens d'ancrage.
2. **Hero Section** : Message d'accueil fort avec un bouton d'action direct vers le contact.
3. **À Propos** : Courte présentation de l'agence.
4. **Services** : Présentation sous forme de cartes élégantes avec des icônes représentatives.
5. **Réalisations (Portfolio)** : Grid d'images illustrant les projets menés à bien.
6. **Témoignages** : Retours d'expérience clients.
7. **Contact** : Formulaire interactif et fonctionnel.

## 🚀 Installation et Utilisation Locale

Pour exécuter et tester ce projet sur votre machine locale, suivez ces étapes simples :

1. **Cloner le projet :**
   ```bash
   git clone https://github.com/votre-utilisateur/igotech-portfolio.git
   ```

2. **Accéder au répertoire :**
   ```bash
   cd igotech-portfolio
   ```

3. **Ouvrir le projet :**
   Double-cliquez simplement sur le fichier `index.html` ou ouvrez-le avec l'extension **Live Server** sur VS Code pour bénéficier d'un rafraîchissement automatique en temps réel.
